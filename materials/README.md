# Included material tables

These are the optical-constant tables supplied with the original code package.
The Streamlit app lists each table with the wavelength unit required by that table:

- `Si_Green.txt`: nm, n/k
- `Si_Aspnes.txt`: µm, n/k
- `Si_Palik.txt`: µm, n/k
- `Si_Palik2.txt`: nm, n/k
- `Au_JC.txt`: µm, n/k
- `Ag_JC.txt`: µm, n/k
- `Au_sharck.txt`: nm, n/k

The app never extrapolates beyond the selected material table. Use a custom file to calculate a different spectral range.
